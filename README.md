# Time-Series-Analysis-of-air-raid-alerts-in-Ukraine.

Інтерактивна аналітична система для дослідження статистики повітряних тривог в Україні. Проєкт базується на обробці відкритих даних та дозволяє візуалізувати динаміку безпекової ситуації в країні.

## 📋 Що входить у проєкт
- **Динаміка тривог:** Історичний графік частоти оголошення тривог по місяцях.
- **Навантаження:** Сумарна тривалість тривог у годинах для оцінки впливу на інфраструктуру.
- **Гео-рейтинг:** Аналіз інтенсивності тривог у розрізі областей України.
- **Теплові карти:** Визначення "найнебезпечніших" годин та днів тижня для окремих регіонів.
- **Сезонність:** Аналіз розподілу тривог за порами року.

## 🚀 Як користуватися
1. Завантажте файл `.ipynb` у Google Colab.
2. **Першочергово** запустіть блок "Підготовка даних" (завантажує та очищує дані).
3. Запускайте наступні блоки аналізу залежно від того, який зріз даних вас цікавить.
4. Для локальної аналітики (теплові карти/тренди) змініть назву області у відповідній змінній `region = '...'`.

## 🛠 Технологічний стек
- **Мова:** Python
- **Аналіз даних:** `pandas`
- **Візуалізація:** `matplotlib`, `seaborn`
